# ePortfolios

**ePortfolio & Competency Platform**

By [Academe Research, Inc](https://eportfolios.in)

---

## About

ePortfolios is an open-source ePortfolio and competency assessment platform
for educational institutions. It provides learners with a personal space to
collect, curate, and share evidence of their learning and professional
development.

## Features

- Personal portfolio pages and collections
- Competency-based assessment frameworks
- File management and multimedia support
- Social networking and group collaboration
- Institution and role management
- Extensible plugin architecture

## Requirements

- PHP 8.1+
- MariaDB 10.6+ or PostgreSQL 12+
- Apache 2.4+ or Nginx
